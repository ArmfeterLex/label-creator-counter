# Label Creator Counter

Приложение на C# Windows Forms для создания меток с заданными параметрами.

## Функциональность

- Ввод текста метки
- Ввод координат X и Y
- Ввод размера (ширина и высота одинаковые)
- Создание метки с рамкой (BorderStyle.FixedSingle)
- Подсчет маленьких меток (размер < 50)
- Подсчет больших меток (размер ≥ 50)
- Отображение счетчика в заголовке окна

## Интерфейс

- textBox1 — текст метки
- textBox2 — координата X
- textBox3 — координата Y
- textBox4 — размер (px)
- button1 — создать метку

## Пример работы

Ввод: текст "Привет", X=100, Y=100, размер=40
Результат: создаётся метка 40×40 с текстом "Привет", маленьких меток +1

Ввод: текст "Большая", X=200, Y=200, размер=80
Результат: создаётся метка 80×80, больших меток +1

## Куда можно встроить

- В визуальные редакторы
- В демонстрацию создания элементов
- В примеры валидации ввода
- В обучающие проекты

## Требования

- Windows
- .NET Framework / .NET Core с WinForms

## Примечание

При неполном заполнении полей кнопка не создаёт метку. Координаты и размер должны быть целыми числами. Все метки имеют одинаковый размер по ширине и высоте.
